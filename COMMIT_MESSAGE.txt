굴삭기 6축 PID/FF 게인 자동 추정 시스템 구축

## 프로젝트 개요
- 165개 CSV 데이터 분석을 통한 6축 PID/FF 게인 추정
- 대상: Arm In/Out, Boom Up/Down, Bucket In/Out
- 목표: τ=2s 기준 PID, 속도 기반 FF 룩업 테이블

## 주요 구현 단계

### 1단계: 통합 분석 시스템 (V3)
- 축/방향별 배치 파싱 (165개 파일 → 6개 그룹)
- 1차 시스템 모델 피팅
- 정상 상태 속도 추출
- FF 게인 통계 산출

### 2단계: 개별 샘플 PID 계산
- 각 파일별 독립 모델 피팅 및 IMC 튜닝
- Duty 부호 처리 (In/Down 음수화)
- K 제약 수정 (K≥0 → K∈R)
- 품질 필터링 (R²≥0.5)

### 3단계: 통계적 게인 추정
- 중앙값 기반 대표 게인 산출
- 평균 성공률: 82.7%
- 평균 모델 R²: 0.92

### 4단계: τ=2s 재계산
- τ=100s(포화) → τ=2s(현실적)
- Kp 불변, Ki 50배 증가
- 정착시간: 400s → 8s

### 5단계: 추가 분석
- 속도별 FF 룩업 테이블 (Boom_Down R²=0.99)
- 조건별 게인 차이 (Single/Couple, High/Low)
- 예상 오차 분석 (duty 오차 10.4%)

### 6단계: 최종 권장 게인 도출
- Phase 1: τ=2s PID + 통합 FF (기본)
- Phase 2: 속도별 FF 룩업 (성능 개선)
- C++ 구현 코드 제공

## 핵심 성과
✅ 6축 전체 PID/FF 게인 추정 성공
✅ 모델 R²=0.924 (우수)
✅ FF R²=0.709 (양호)
✅ 정착시간 8초 (실용적)
✅ 즉시 적용 가능한 게인 도출

## 주요 파일
- src/: 핵심 모듈 (parser, identification, tuning)
- scripts/: 분석 스크립트 (V3, 후처리, τ=2s 재계산)
- output/: 분석 결과 (게인, 통계, 시각화)
- FINAL_GAINS_SUMMARY.md: 최종 게인 종합 문서 (15개 섹션)
- run_all_final.bat: 전체 파이프라인 자동 실행

## 기술 스택
- Python 3.11
- pandas, numpy, scipy (모델 피팅)
- matplotlib, seaborn (시각화)
- IMC 튜닝 (PID 게인 계산)
